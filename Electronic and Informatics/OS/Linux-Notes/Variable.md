# Variable

Dalam bash kita dapat mengoperasikan variabel.

## Temporary variable

Variabel yang hanya tersedia dalam satu terminal saja, ketika ditutup maka hilang, kita dapat membuatnya dengan menggunakan:

```bash
~ name="joe"
~ echo $name
```

## Global Variable

Variabel yang tersedia secara global pada OS, akan tetap ada walaupun OS direstart.
