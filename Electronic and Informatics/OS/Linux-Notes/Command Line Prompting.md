# Penjelasan

Pada OS linux, terdapat banyak command yang berguna untuk melakukan suatu hal dengan cepat. Selain itu, biasanya server menggunakan OS linux tanpa GUI, karena dapat menghemat RAM dan meningkatkan performa server. Tetapi, tidak bisa kita pungkiri bahwa kita terbiasa menggunakan GUI yang juga dapat meningkatkan pengalaman dalam mengoperasikan suatu hal basic dengan mudah. Namun, penggunaan command-line akan memberikan kontrol yang lebih advance dan luas lagi untuk meningkatkan efisiensi kita.

## Contoh script untuk time-consuming Task
- ```for i in {1..50}; do touch "index$i.html"; done;```
- selector file dengan regex/pattern name
