# Penjelasan

| Perintah                   | Fungsi                                   |
| -------------------------- | ---------------------------------------- |
| `command > /dev/null`      | Hilangkan output biasa (stdout)          |
| `command 2> /dev/null`     | Hilangkan pesan error (stderr)           |
| `command > /dev/null 2>&1` | Hilangkan semua output (stdout + stderr) |
