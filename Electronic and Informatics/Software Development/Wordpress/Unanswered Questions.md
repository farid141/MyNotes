# Unanswered Questions

Jadi dari mana saja kita dapat mengedit tampilan CMS wordpress, saat ini saya tahu beberapa:

- wp-admin bagian appearance saya lihat bisa edit menu navbar
- bagian customize page dengan blog/elementor editor
- dari API js sebagai http client?

apakah elementor sepenuhnya menggantikan blog editor (semua fitur blog pasti ada di elementor)?

apa bedanya themes dan plugin

wordpress bisa menggunakan icon library untuk mendapatkan logo

## Form

kita dapat membuat form dengan menggunakan WPForms, setiap form emiliki shortcode yang dapat digunakan pada template

## Menambahkan halaman list of blog

1. Pergi ke wp-admin->settings->reading
2. Atur posts page ke halaman yang dituju (otomatis konten akan berisi list of blog)

## Untuk membuat data post

wp-admin->posts->add new posts
