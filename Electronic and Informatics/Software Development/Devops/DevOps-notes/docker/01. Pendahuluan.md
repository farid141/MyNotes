# Pendahuluan

docker engine berisi aplikasi client-server:

- server berjalan dengan daemon dockerd
- API untuk berkomunikasi dengan docker daemon
- CLI sebagai client docker

## hal yang dimanage docker

1. image
2. container, berjalan di atas image

bisa dipasang ke container
3. volume
4. network

## Inspect dapat digunakan pada

1. image (perintah apa yang digunakan, Environment variable apa yang digunakan, Atau port apa yang digunakan)
2. container (Volume apa yang digunakan, Environment variable apa yang digunakan, Port forwarding apa yang digunakan)

prune, membersihkan image/container/volume/network

## Direktori Docker

Jika menggunakan docker pada linux, kita dapat melihat direktori yang berisi komponen docker

| Direktori     | Fungsi                                                                                    |
| ------------- | ----------------------------------------------------------------------------------------- |
| `containers/` | Menyimpan data setiap container (log, metadata, dll).                                     |
| `image/`      | Metadata image Docker yang digunakan (bisa berbeda tergantung storage driver).            |
| `overlay2/`   | Layer filesystem dari container & image (jika memakai `overlay2` sebagai storage driver). |
| `volumes/`    | Data volume Docker yang persisten.                                                        |
| `network/`    | Informasi konfigurasi jaringan container.                                                 |
| `plugins/`    | Plugin eksternal Docker.                                                                  |
| `swarm/`      | Data terkait mode swarm (orchestrasi container).                                          |
| `builder/`    | Data hasil build image lokal.                                                             |
| `runtimes/`   | Konfigurasi runtime (misal `runc`).                                                       |
| `tmp/`        | File sementara saat menjalankan container atau build image.                               |
