# Docker for Development

di development kita harus bisa mengubah code dan mungkin menambah library

Sedangkan untuk development, tidak bisa karena seluruh code sudah dipindah ke container, kalau mau kita bisa masuk ke container dan memodifikasi, namun setelah container restart, modifikasi akan hilang

docker image bisa saja tetap sama dengan prod, tetapi pada development, gunakan bindmount directory host ke container agar COPY . . di dockerimage teroverwrite.
