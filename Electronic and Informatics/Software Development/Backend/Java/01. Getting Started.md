# Getting Started

Instalasi Java pergi ke oracle, akan terdownload JDK dan compiler. Selain itu juga include `jshell` untuk menjalankan code java dalam terminal.

`JDK` Java Development Kit: untuk compile code
`JRE` Java Runtime Environment: Berisi library yang dibutuhkan
`JVM` Java Virtual Machine: untuk menjalankan hasil compile

## Basic Project

Sebuah projek harus memiliki file java dan main method.

`public static void main (string [])`

Untuk menjalankan code java:

1. compile `javac Hello.java`, akan menghasilkan `.class`
2. jalankan `java Hello`

Access Modifier, membatasi sebuah variabel, class, fungsi untuk diimport dari file lain

apa ngga berat kalo eksekusi hasil compiler harus pake JVM, bukan binarry seperti golang
