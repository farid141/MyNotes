# Spring JDBC

Java Database Connection, merupakan library/dependency untuk mempermudah menghubungkan app dengan DB (dibandingkan dengan JDBC regular)

JDBC secara otomatis mengeksekusi 2 file sql ketika app dimulai:
`app/resources/schema.sql`
`app/resources/data.sql`

H2 Database

Merupakan DB in-Memory yang akan terhapus setelah aplikasi mati. `Jdbc template` berisi konfigurasi DB, secara default menggunakan H2

Memanfaatkan tools IDE untuk coding:

- `buat class` ctrl+shift+p + create class
- `generate setter getter` klik kanan pada editor + generate setter and getter
- `buat method sebuah class` panggil method + create method on file
- `pindah folder` ubah nama package + move file into ...
