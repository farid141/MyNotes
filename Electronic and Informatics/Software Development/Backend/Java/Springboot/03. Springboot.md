# Spring Boot

Dalam springboot terdapat embedded server yang otomatis jalan.

## Annotations

Untuk mengelola IoC dalam springboot, terdapat beberapa annotations yang dapat digunakan:

- `@Controller`
- `@Service`
- `@Repository` Otomatis handling DB exceptions
- `@RestController` combine `@Controller` and `@ResponseBody`

Semua annotations tersebut sebenarnya adalah `@Component` yang ditambahkan beberapa annotations lagi didalamnya
