# Operation and Data Type

Operasi yang bisa digunakan:

```bash
# mengubah string value dari key
set key "value"
# mendapatkan value menggunakan key, return (nil) jika kosong
get key
# mengecek apakah key memiliki value (return bool), lebih hemat dibanding "get"
exists key
# menghapus menggunakan key, (bisa multiple)
del key [key ...]
# menambah data value ke key
append key value
# search key by patterns (tidak direkomendasikan karena proses scanning berat)
keys pattern/regex

# Memodifikasi / ambil sebagian 
# replace value dari offset yang ditentukan
setrange key offset "tes"
# mengambil value substring dari key
getrange key start end

# ambil nilai dari beberapa key sekaligus
mget key [key ...]
# set multiple key-value sekaligus 
mset key value [key value ...]
```

## Expiration

Sebuah key bisa memiliki masa expired:

- `-2` expired
- `-1` not set

```bash
# set expiration of existing key
expire key seconds

# set value and expiration of key
setex key seconds value

# get expiration of key
ttl key
```

## Increment & Decrement

Mencegah race condition dari:

1. get nilai key
2. set nilai key+1

```bash
incr key
decr key
incrby key val
decrby key val
```

## Flush

Untuk menghapus semua key di db aktif atau semua db secara cepat.

```bash
# hapus semua keys pada semua db
flushall
# hapus data pada db sekarang
flushdb
```
