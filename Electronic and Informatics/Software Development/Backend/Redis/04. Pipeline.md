# Pipeline

Digunakan untuk migrasi data secara masif dari db storage ke redis. Daripada mengirim command satu persatu. Namun kita `tidak akan mengetahui balasan tiap barisnya`.

## Perintah

`redis-cli -h host -p port -n database --pipe < file_input`

File input berisi `baris command redis`

Contoh diatas adalah menggunakan `redis-cli` tapi bisa juga dengan bahasa pemrograman.
