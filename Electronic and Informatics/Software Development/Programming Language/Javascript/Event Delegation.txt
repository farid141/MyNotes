
event delegation: 
Dalam skenario penambahan event terhadap sebuah komponen, kita perlu menambahkan event listener kemudian menggunakan:
1. vanilla javascript: 
document.addEventListener(event, function(e){
	if(e.target.matches(selector)){
		callback(e);
	}
})
