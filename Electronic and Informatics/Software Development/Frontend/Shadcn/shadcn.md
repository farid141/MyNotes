# ShadcnUI

shadcnUI lebih kustomable dibanding chakraUI dan materialUI karena kita mendapatkan salinan komponen pada folder projek kita.

## Customisasi Komponen

secara bawaan shadcn sudah menggunakan tools `tailwindCSS` dan `CVA` mengkustomnya dibuthukan basic understanding dari:

- tailwindcss
- class-variance-authority js library
