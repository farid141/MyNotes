# Props

Prop adalah parameter yang diberikan ke komponen saat pemanggilan komponen seperti `atribut elemen HTML`, bisa berisi `variable/function/jsx`. Kemudian ada `special props` yaitu `children` yang merupakan bagian diantara tag `pembuka dan penutup`. Berikut adalah contoh2 implementasinya.

## Children

Jika komponen sebagai wrapper, kita dapat pass special props 'children'

```jsx
<Komponen>
{children}
</Komponen>
```

## Props destructuring

Sebuah komponen mengambil variabel lain dan mengumpulkannya dalam satu object

```jsx
export default function Komponen({children, ...props}){
 return (
  <div {props}></div>
 )
}
```

## Passing wrapper

pada saat memanggil komponen:

1. built-in element: ButtonWrapper="menu"
2. custom element: ButtonWrapper={Section}

## Lifting state up

Merupakan sebuah teknik untuk mengupdate state parent tetapi dengan aksi yang dilakukan/ditrigger dari child. Dilakukan dengan `mempassing fungsi` dari parent ke child. Fungsi tersebut, `berisi setter state` dalam parent.
