# Remote Working

Untuk bekerja secara remote dengan perusahaan luar negeri, administrasi akan bergantung dimana tempat pekerja bekerja.

- Remote di negara Pemberi Kerja
- Remote di negara pekerja (pajak penghasilan luar negeri dan skema kontrak)
- Remote di negara ketiga (butuh visa bekerja di negara ketiga)
